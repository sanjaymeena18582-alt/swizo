# E-Commerce Website Documentation

## Features
- **User Authentication:** Secure login and registration for users.
- **Product Catalog:** Browse through a wide range of products with filtering and sorting options.
- **Shopping Cart:** Add products to a cart and manage items before checkout.
- **Checkout Process:** Seamless checkout with payment integration.
- **Order History:** View past orders and track current orders.
- **Admin Dashboard:** Manage products, orders, and users from an admin interface.

## Setup Instructions
1. **Clone the Repository**  
   `git clone https://github.com/sanjaymeena18582-altCreate/swizo.git`
2. **Install Dependencies**  
   Navigate to the project directory and run:  
   `npm install`  
3. **Set Up Environment Variables**  
   Create a `.env` file in the root directory and set the required variables as per the provided example.
4. **Run the Application**  
   Use the following command to start the application:  
   `npm start`

## Usage Guide
- Open your web browser and navigate to `http://localhost:3000`.
- Use the navigation bar to access different features of the website.
- Register a new account or login to an existing account to access all functionalities.
- Browse products and add items to your shopping cart.
- Proceed to checkout to complete your purchases.
- Admins can log in to manage the website features.
